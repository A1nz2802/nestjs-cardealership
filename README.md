## Overview

- Build a REST API with NestJS
- Modules 
- Controllers (Get, Post, Patch, Delete)
- Services
- Decorators
- Dependency injection
- Pipes
- Exception Filters
- DTO (Data Transfer Object)
- Class Validator & Class Transformer
- DRY principle
- Nest CLI usage
- SEED endpoint
- Local data

## Endpoints

- Cars Module
```bash
# Get All Cars
http://localhost:3000/cars
# Get Cars By Id
http://localhost:3000/cars/<uuid>
# Post Car
http://localhost:3000/cars
# Update Car
http://localhost:3000/cars/<uuid>
# Delete Car
http://localhost:3000/cars/<uuid>
```

- Brands Module
```bash
# Get All Brands
http://localhost:3000/brands
# Get Brands By Id
http://localhost:3000/brands/<uuid>
# Post Brand
http://localhost:3000/brands
# Update Brand
http://localhost:3000/brands/<uuid>
# Delete Brand
http://localhost:3000/brands/<uuid>
```
